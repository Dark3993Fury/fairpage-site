# FairPage

FairPage is a free Chrome extension that detects dark patterns and checkout traps on the web. It actively scans pages locally to find hidden fees, fake urgency, sneaky opt-ins, and subscription traps before they cost you money.

## Running Locally
This website is completely static. To run it locally:
1. Clone the repository.
2. Open `index.html` in your web browser. 

No build steps or dependencies required.

## Deployment to GitHub Pages
1. Push this repository to GitHub.
2. Go to the repository **Settings**.
3. Navigate to **Pages** in the left sidebar.
4. Under **Source**, select the main branch and root folder.
5. Save, and your site will be deployed shortly.
